Planificación

En esta etapa planifiqué (¿?) mi proyecto personal. Para ello llevé a cabo distintas acciones que delimitaron mi investigación posterior y facilitaron la elaboración de mi producto y la recolección de información.

Objetivo

Aprender a crear un sitio web ideal de comercio electrónico, mediante el uso de lenguajes de programación HTML 5, CSS y JavaScript en Visual Studio Code.

Producto

Sitio web que concentra diversas herramientas prácticas y recomendaciones que permite a pequeños emprendedores, que no cuentan con recursos suficientes para contratar a profesionales, crear su propio sitio web para la promoción y/o venta de bienes y/o servicios a través de comercio electrónico.

Criterios de Logro

Mi producto puso especial énfasis en las características esperadas por mi público objetivo, las mismas que fueron relevadas a través de una encuesta. Por ello, considero que es un medio que les permite contar, de manera gratuita, con recomendaciones y herramientas prácticas para adquirir el conocimiento necesario y crear sus sitios web sin depender de profesionales en la materia.
